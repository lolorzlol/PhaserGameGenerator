## 课件技术规格

| 项目 | 规格 |
|------|------|
| 风格 | Soft Game（柔和粉彩 + 游戏几何元素） |
| 总页数 | 13 页幻灯片 |
| 尺寸适配 | 100vh viewport 自适应，支持 clamp() 缩放 |
| 导航方式 | 键盘（方向键/空格）、点击导航点、触摸滑动 |
| 编辑功能 | 浏览器内编辑（E键切换），Ctrl+S 导出 |

## 动画规范

### 入场动画（reveal）
```css
opacity: 0;
transform: translateY(25px) scale(0.98);
transition: opacity 0.6s var(--ease-out), transform 0.6s var(--ease-out);
```

### 延迟序列
- 第1个子元素：0.05s
- 第2个子元素：0.15s
- 第3个子元素：0.25s
- 第4个子元素：0.35s
- 第5个子元素：0.45s

### 几何浮动
```css
animation: floatShape 10s ease-in-out infinite;
/* 0% 100%: translate(0,0) scale(1) */
/* 50%: translate(15px,-25px) scale(1.05) */
```

### 减少动画支持
```css
@media (prefers-reduced-motion: reduce) {
  .geo-shape { animation: none; }
  .reveal { transition-duration: 0.2s; transform: none; }
}
```
