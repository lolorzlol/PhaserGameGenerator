# Trae 安装与基本配置

## 1. Python 安装

1. 从 [python.org](https://www.python.org/downloads/) 下载 Python 3.10+
2. 安装时务必勾选 **"Add Python to PATH"**
3. 验证安装：

```bash
python --version
pip --version
```

## 2. Node.js 安装

1. 从 [nodejs.org](https://nodejs.org/) 下载 **LTS 版本**（v20+ 推荐）
2. 使用默认选项完成安装
3. 验证安装：

```bash
node --version
npm --version
```

## 3. Git 安装

1. 从 [git-scm.com](https://git-scm.com/downloads) 下载并安装 Git
2. 安装后验证：

```bash
git --version
```

## 4. Trae IDE 安装

1. 从 [trae.ai](https://www.trae.ai/) 下载并安装 Trae
2. 首次打开后登录账号
3. 在设置中确认 AI 助手功能正常


## 5. 安装依赖

powershell中执行：

```bash
# 核心依赖
npm install phaser

# 开发依赖
npm install -D typescript @types/node vite
```

# Playwright MCP 安装（可选）

1. 确保已安装 Node.js（v18+），可从 https://nodejs.org/ 下载。

2. 全局安装 Playwright MCP：

```bash
npm install -g @anthropic/playwright-mcp
npm install -g @playwright/test
```

3. 安装 Chromium：

```bash
npx playwright install chromium
```

4. 验证安装：

```bash
npx @anthropic/playwright-mcp --help
```

5. 配置到 Claude Code / Trae：

在 `~/.claude/settings.json` 的 `mcpServers` 中添加：

```json
{
  "mcpServers": {
    "playwright": {
      "command": "npx",
      "args": ["@anthropic/playwright-mcp"]
    }
  }
}
```

6. 重启 Claude Code / Trae 使配置生效。

## 2. 复制配置文件

将项目根目录的 `TraeConfigs/ProjectConfigs/` 复制到项目根路径下，使 Trae 能够加载 AI 行为配置：

```bash
cp -r /path/to/Phaser-Game-Generator/TraeConfigs/ProjectConfigs .trae/
```

## 3. 用 Trae 打开项目

在 Trae 中打开项目文件夹，确认 AI 助手已加载 `AGENTS.md` 中的配置。

## 4. 用 Superpowers 的 Brainstorming 开始

在 Trae 的 AI 对话框中输入游戏概念，触发 brainstorming skill，例如：

> "我想做一个塔防小游戏，有塔放置、敌人波次、攻击系统"

AI 会引导你完成需求澄清 → 设计文档 → 实施计划 → 代码生成的完整流程。
