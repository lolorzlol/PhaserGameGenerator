# Playwright MCP安装
  1. 安装 Node.js

  确保已安装 Node.js（v18+），可从 https://nodejs.org/ 下载。

  2. 全局安装 Playwright MCP

  npm install -g @anthropic/playwright-mcp

  npm install -g @playwright/test

  3. 安装chrome浏览器和命令行安装chromium

  npx playwright install chromium

  4. 验证安装

  npx @anthropic/playwright-mcp --help

  5. 配置到 Claude Code

  在 ~/.claude/settings.json 的 mcpServers 中添加：

  {
    "mcpServers": {
      "playwright": {
        "command": "npx",
        "args": ["@anthropic/playwright-mcp"]
      }
    }
  }

  6. 重启 Claude Code
