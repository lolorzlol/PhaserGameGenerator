# Trae 安装与基本配置

## 1. Python 安装

1. 从 [python.org](https://www.python.org/downloads/) 下载 Python 3.11以上版本
2. 安装时务必勾选 **"Add Python to PATH"**
3. 验证安装：

```bash
python --version
pip --version
```

## 2. Node.js 安装

在 PowerShell 中执行以下命令，使用 winget 安装 LTS 版本（v20+）：

```powershell
winget install OpenJS.NodeJS.LTS
```

安装完成后**重启 PowerShell**，然后验证：

```bash
node --version
npm --version
```

## 3. Git 安装

1. 从 [git-scm.com](https://git-scm.com/downloads) 下载并安装 Git
2. 安装后验证：

```bash
git --version
```

## 4. Trae IDE 安装

1. 从 [trae.ai](https://www.trae.ai/) 下载并安装 Trae
2. 首次打开后登录账号
3. 在设置中确认 AI 助手功能正常


## 5. 安装依赖

powershell中执行：

```bash
# 核心依赖
npm install phaser

# 开发依赖
npm install -D typescript @types/node vite

npm install create-vite@9.0.6

```

# Playwright MCP 安装（可选）

1. 确保已安装 Node.js（v18+），可从 https://nodejs.org/ 下载。

2. 全局安装 Playwright MCP：

```bash
npm install -g @anthropic/playwright-mcp
npm install -g @playwright/test
```

3. 安装 Chromium：

```bash
npx playwright install chromium
```

4. 验证安装：

```bash
npx @anthropic/playwright-mcp --help
```

5. 配置到 Claude Code / Trae：

在 `~/.claude/settings.json` 的 `mcpServers` 中添加：

```json
{
  "mcpServers": {
    "playwright": {
      "command": "npx",
      "args": ["@anthropic/playwright-mcp"]
    }
  }
}
```

6. 重启 Claude Code / Trae 使配置生效。
