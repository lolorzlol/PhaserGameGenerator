首先需要说明的是，小游戏最常用的是 Cocos Creator 这样的引擎。这里用的技术栈是 Phaser + TypeScript + Vite 的 H5 小游戏，两者有相似之处，例如都用 TypeScript / JavaScript 语言。

虽然用 Unity 也能做小游戏，但是没有上面的用的多。小游戏用一些更简单的开发技术栈效率更高。

# 环境安装与基本配置

> 环境安装前先说明 PowerShell 的打开方式，后续步骤中会经常使用。
> 点击底部 **开始菜单（Windows 图标）**，键盘搜索 `powershell`，点击打开即可（Win10 同理）。
>
> <img src="images/image-11.png" width="50%" />

## 1. Python 安装

1. 从 [python.org](https://www.python.org/downloads/) 下载 Python 3.11 以上版本
2. 安装时务必勾选 **”Add Python to PATH”**
3. 验证安装，在 PowerShell 中运行：

```powershell
python --version
pip --version
```

## 2. Node.js 安装

在 PowerShell 中执行以下命令，使用 winget 安装 LTS 版本（v20+）：

```powershell
winget install OpenJS.NodeJS.LTS
```

安装完成后**重启 PowerShell**，然后验证：

```powershell
node --version
npm --version
```

## 3. Git 安装

安装网址：[git-scm.com/install/windows](https://git-scm.com/install/windows)

选择下载 Windows x64 的版本：

![alt text](images/image-19.png)

运行安装包：

![alt text](images/image-20.png)

**检查是否安装成功**，在 PowerShell 中输入：

```powershell
git --version
```

## 4. Trae IDE 安装

1. 从 [trae.ai](https://www.trae.ai/) 下载并安装 Trae
2. 首次打开后登录账号
3. 在设置中确认 AI 助手功能正常

# Playwright MCP 安装（可选）

1. 确保已安装 Node.js（v18+）

2. 全局安装 Playwright 相关包：

```powershell
npm install -g @playwright/test
```

3. 安装 Chromium：

```powershell
npx playwright install chromium
```
